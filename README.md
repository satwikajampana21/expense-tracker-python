# Expense Tracker and Analysis

This project is a Python-based expense tracker that helps analyze personal spending patterns and generate insights using data analysis techniques.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Features

* Recording and tracking expenses
* Categorizing spending data
* Generating summaries of expenses
* Visualization of spending patterns

## Project File

expense_tracker.ipynb

## Purpose

The goal of this project is to practice data analysis and build a simple system to track and analyze personal expenses.
